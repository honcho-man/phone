Web Design Of Phones Using HTML, CSS and JS.

This project involves designing of phones (old mobile phones and smartphones) using just HTML, CSS and a bit of JavaScript to give the phones some functions like switching on and other things.



DEMOS

I had this idea while i was going through twitter checking posts that used #100daysofcode and came across a tweet where someone used just HTML and CSS to design a NOKIA 1280 and i was so amazed and also inspired. So i did a replica of what he did but made it function where it works like a real phone (not all the functions though) and it really got alot of reviews on twitter and that encouraged me to start this project.

Well i hope you enjoy it!
